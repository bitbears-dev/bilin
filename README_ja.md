# Bilin

> **[Read in English 🇬🇧/🇺🇸](./README.md)**

Bilin は、英語で書かれた長文の海外ニュースやブログ記事を、語学学習も兼ねてスムーズに読み進めるために開発された AI 搭載型のデスクトップアプリケーションです。
オリジナルの Web サイトと、段落ごとに抽出された対訳カードを並列（Side-by-Side）で表示でき、読み上げ機能にも対応しています。

## 🚀 主な機能

- **並列表示 (Side-by-Side) レイアウト**: 中央ペインに本物の Web サイトのオリジナルビューを残したまま、右ペインの抽出されたカードで学習を進めることができます。
- **段落ごとの AI 翻訳**: OpenAI API（`gpt-4o-mini`）を活用し、文脈に沿った自然な日本語訳をパラグラフ単位で必要な分だけ表示します。
- **ネイティブ音声の読み上げ (TTS)**: 各段落の「Listen」ボタンを押すことで、指定した英語の自然な読み上げ音声を生成し、リスニング学習に活用できます。
- **ローカルキャッシュでの節約**: 一度翻訳したテキストや取得した音声データは、自動的に IndexedDB に保存されます。再度学習する際の API 呼び出しコストや待機時間はゼロになります。
- **複数タブ (セッション) 管理**: アプリを閉じても状態が維持されるセッション一覧に対応しており、「あとで読む」用途でも途切れることなく学習を再開できます。
- **BYOK (Bring Your Own Key) 方式**: 翻訳や音声生成に必要な API キーはご自身のものを入力頂きます。キーはデバイスのローカルストレージにのみ保存され、安全です。

## 📦 インストール方法

現在の環境に導入するための主な2つの手段をご用意しています。

### Option 1: ビルド済みアプリのダウンロード（推奨）
本アプリは GitHub Actions を用いて、リリース毎に各 OS 向けの実行ファイルを自動で生成しています。
1. プロジェクトの [Releases タブ](../../releases) へアクセスします。
2. お使いの OS に合わせたファイルを選んでダウンロードします。
   - **Windows**: `.msi` または `.exe` ファイル
   - **macOS**: `.dmg` ファイル
   - **Linux**: `.AppImage` または `.deb` パッケージ
3. ダウンロードしたファイルを開いてインストールを実行します。

### Option 2: ソースからのビルド（開発者向け）
ご自身の環境でソースコードから直接アプリをビルドする場合の手順です。実行には、**Node.js** および **Rust Toolchain** のインストール環境が必要です。

```bash
# 1. リポジトリをクローンする
git clone https://github.com/bitbears-dev/bilin.git
cd bilin

# 2. 依存パッケージのインストール
npm install

# 3. 開発用サーバー (Tauri) でアプリを起動
npm run tauri dev

# 4. (任意) お使いのOS用のアプリ実行バイナリをビルド
npm run tauri build
```

## 📖 基本的な使い方

1. **APIキーの設定**: 画面の左下にある **設定（歯車アイコン）** をクリックし、ご自身の OpenAI API キーを入力して「保存」します。
2. **記事を開く**: 画面上部の URL 入力バーに、読みたい英語のニュースや記事等のリンクを貼り付け、**「開く」** を押します。
3. **読んで学ぶ**: 記事の本文が抽出され、右側のパネルにパラグラフごとのカードになって表示されます。
   - 意味がわからないところは **「翻訳を表示」** をクリックします。
   - 発音を確認したいときは **「Listen」** をクリックします。
4. **タブの管理**: 別のサイトも見たい場合は、左側の垂直タブにある **「＋」アイコン** を押すことで、現在の状態を保持したまま新しい空のタブを作成できます。
